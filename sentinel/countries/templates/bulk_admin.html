Data Dictionary

code
The ISO 3166 alpha-2 country code for the country.
name
The name of the country as you want it displayed.
mdr_estimated_cases
Number of estimated cases of MDR-TB for the country.
mdr_eval_0
Number of 0-4 year-olds needing evaluation
mdr_eval_5
Number of 5-14 year-olds needing evaluation
mdr_treat_0
Number of 0-4 year-olds needing treatment
mdr_treat_5
Number of 5-14 year-olds needing treatment
mdr_therapy_0
Number of 0-4 year-olds needing preventative therapy
mdr_therapy_5
Number of 5-14 year-olds needing preventative therapy
reported_mdr
Has there been a reported case of MDR-TB? (0 = False, 1 = True)
reported_xdr
Has there been a reported case of XDR-TB? (0 = False, 1 = True)
documented_adult_mdr
Has there been a documented case of MDR-TB among adults? (0 = False, 1 = True)
documented_adult_xdr
Has there been a documented case of XDR-TB among adults? (0 = False, 1 = True)
documented_child_mdr
Has there been a documented case of MDR-TB among children? (0 = False, 1 = True)
documented_child_xdr
Has there been a documented case of XDR-TB among children? (0 = False, 1 = True)
